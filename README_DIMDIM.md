# Dimdim (web)

## Rodar local
1. Copie `.env.local.example` para `.env.local` e preencha:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
2. Instale e rode:

```bash
npm install
npm run dev
```

Abra http://localhost:3000

## Rotas
- /login
- /signup
- /app/dashboard

> Observação: este pacote não inclui `node_modules`.
